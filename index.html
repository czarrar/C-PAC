Discovery Science for the Human Connectome

Researchers can now explore functional connectome data from thousands of participants made public by the International Neuroimaging Data-sharing Initiative (INDI) and its 1000 Functional Connectomes Project. Beyond access to data, scientists need appropriate tools that facilitate data exploration even for those who are inexperienced with fMRI image analysis or lack the programming support necessary for handling and analyzing large data sets.

The Configurable Pipeline for the Analysis of Connectomes (C-PAC)

A configurable, open-source, Nipype-based, automated processing pipeline for resting state fMRI data. Designed for use by both novice users and experts, C-PAC brings the power, flexibility and elegance of Nipype to users in a plug-and-play fashion; no programming required.

Novice-Friendly Automation and Scalability

Users can rapidly orchestrate automated pre-processing and data analysis for multiple subjects by modifying an easy to read, text-editable configuration file.

  • Run on locally or on a compute cluster using Sun Grid Engine or the Portable Batch System.
  • No need to re-arrange your files; C-PAC adapts to most existing directory structures.
  • Forkable analysis pipelines allow users to run C-PAC using multiple settings at once.
  • Easily explore the impact of preprocessing decisions on your findings.

Preprocessing Features

  • Motion Correction
  • Coregistration
  • Normalization
  • Spatial Smoothing
  • Temporal Filtering
  • Segmentation
  • Nuisance Signal Regression
  • Median Angle Correction
  • Motion Scrubbing

Measures and Outputs

  • Seed-based Correlation Analysis
  • Amplitude of Low Frequency Fluctuations (ALFF) and fALFF
  • Regional Homogeneity
  • Voxel-Mirrored Homotopic Connectivity
  • Timeseries Extraction
  • Network Centrality
  • Connectome-Wide Association Studies
  • Bootstrap Analysis of Stable Clusters
  • Group-level Statistics
