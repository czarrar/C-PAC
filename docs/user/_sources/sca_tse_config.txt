Seed-based Correlation Analysis and Timeseries Extraction
---------------------------------------------------------
Seed-based Correlation Analysis Options
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
runSCA
""""""
Set to 1 to run Seed-based Correlation Analysis, set to 0 to skip::

	# Run Seed-based Correlation Analysis
	runSCA = [1]

SCA will be run on all ROI and voxel timeseries extracted below. This means that in order to run SCA, you must also run Timeseries Extraction. Seeds for SCA must be specified in :command:`roiDirectoryPath` or :command:`maskDirectoryPath`.

Timeseries Extraction Options
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
runROITimeseries
""""""""""""""""
Extract an average timeseries for each ROI specified by :command:`roiDirectoryPath`. Required if you wish to run ROI-based SCA. Set to 1 to run, set to 0 to skip::

	runROITimeseries = [0]

roiTSOutputs
""""""""""""
Specify how to output ROI timeseries data::

	# Export ROI timeseries data
	# First value = Output .csv
	# Second value = Output numPy array
	# Options are True/False
	roiTSOutputs = [True, True]

roiDirectoryPath
""""""""""""""""
Specify the full path to a directory containing ROI definitions::

	# Directory containing ROI definitions
	roiDirectoryPath = '/path/to/roi_definitions_directory'

runVoxelTimeseries
""""""""""""""""""
Extract timeseries data for all individual voxels within a mask specified by :command:`maskDirectoryPath`. Required if you wish to run voxel-based SCA. Set to 1 to run, set to 0 to skip::

	runVoxelTimeseries = [1]

voxelTSOutputs
""""""""""""""
Specify how to output voxel timeseries data::

	# Export voxel timeseries data
	# First value = Output .csv
	# Second value = Output numPy array
	# Options are True/False
	voxelTSOutputs = [False, False]

maskDirectoryPath
"""""""""""""""""
Specify the full path to a directory containing masks::

	# Directory contaning masks
	maskDirectoryPath = '/path/to/mask_definitions_directory'

runSurfaceRegistraion
"""""""""""""""""""""
Register timeseries data to a surface model built by FreeSurfer. Required to run vertex timeseries extraction. Set to 1 to run, set to 0 to skip::

	runSurfaceRegistraion = [0]

reconSubjectsDirectory
""""""""""""""""""""""
Specify the directory where FreeSurfer outputs surface data. This should be the same as SUBJECTS_DIR in :file:`.bashrc`::

	reconSubjectsDirectory = '/path/to/fs_output_directory'

runVerticesTimeSeries
"""""""""""""""""""""
Extract timeseries data for surface vertices. Set to 1 to run, set to 0 to skip::

	runVerticesTimeSeries = [0]

verticesTSOutputs
"""""""""""""""""
Specify how to output vertext timeseries data::

	# Export vertex timeseries data
	# First value = Output .csv
	# Second value = Output numPy array
	# Options are True/False
	verticesTSOutputs = [False, False]
