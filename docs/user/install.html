Installing C-PAC¶

System Requirements¶

C-PAC is designed to run in a Unix-like environment such as Linux or Mac OS X, and thus does not support Windows.

[WAITING ON PROGRAMMERS FOR MIN SYS REQS]

Python¶

Python is the glue (Bloom, 2012) that ties all the pieces of C-PAC together. Though many computers come with Python pre-installed, C-PAC relies on a number special-purpose libraries. A complete list of these libraries can be found in the C-PAC developer documentation [ASK DEVS RE: FINDING THIS LIST]

For most users, it is easier to install a pre-compiled Python distribution than to install each library manually. We recommend using the Enthought Python Distribution (EPD). If you are an academic user, you can download EPD for free.

Click here for instructions on how to install EPD.

Nibabel & Nipype¶

C-PAC relies heavily on tools developed as part of the Neuroimaging in Python (NiPy) project. Specifically, Nipype and Nibabel must be installed for C-PAC to run. If you have installed EPD, simply open up a new terminal session and type:

easy_install nibabel
easy_install nipype

This will automatically install both tools. Alernate installation instructions can be found in the NiPy documentation.

AFNI and FSL¶

C-PAC harnesses the power of two leading neuroimaging software packages to perform key analyses.

  • AFNI (which stands for Analysis of Functional NeuroImages) is a set of programs for processing, analyzing and displaying fMRI data (Cox, 2011). Click here for instructions on how to download and install AFNI.
  • FSL (which stands for FMRIB Software Library) is a comprehensive library of analysis tools for FMRI, MRI, and DTI brain imaging data (Jenkinson et al., 2012). Click here for instructions on how to download and install FSL.
