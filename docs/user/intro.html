Features¶

Seed-based Correlation Analysis (SCA)¶

Calculates functional connectivity between a seed region and all other voxels.

Key Paper: Functional connectivity in the motor cortex of resting human brain using echo-planar MRI.

Time-Series Extraction¶

Extract time-series data for pre-defined units, graph vertices, or each voxel.

Voxel-Matched Homotopic Connectivity (VMHC)¶

Key Paper: Growing together and growing apart: regional and sex differences in the lifespan developmental trajectories of functional homotopy.

Regional Homogeneity (ReHo)¶

Measures the similarity of the time series of a given voxel to those of its nearest neighbors.

Key Paper: Regional homogeneity approach to fMRI data analysis.

Bootstrap Analysis of Stable Clusters (BASC)¶

Quantifies the stability of resting-state networks.

Key Paper: Multi-level bootstrap analysis of stable clusters in resting-state fMRI.

Connectome Wide Association Studies (CWAS)¶

Relate a phenotype to every voxel’s pattern of whole-brain functional connectivity.

Key Resource: Connectir

Fractional Amplitude of Low Frequency Fluctuations¶

Key Paper: An improved approach to detection of amplitude of low-frequency fluctuation (ALFF) for resting-state fMRI: fractional ALFF.

Network Centrality¶

Centrality measures quantify functional connectivity within the brain connectome.

Key Paper: Network centrality in the human functional connectome.

FSL Group Analysis¶

Key Resource: FMRI Expert Analysis Tool (FEAT)
